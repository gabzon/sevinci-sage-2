Diseño & Desarrollo


Wordpress

WordPress es un software de internet que usted puede utilizar para crear hermosas páginas web o blogs. Nos gusta decir que WordPress es tanto gratis como valioso al mismo tiempo

Twitter bootstrap

El framework HTML, CSS y JS mas popular para desarrollar proyectos web & mobiles

Jquery

jQuery es una biblioteca de JavaScript rápida, pequeña y rica en funciones. jQuery ha cambiado la forma en la que millones de personas escriben con JavaScript

Font Awesome

Font Awesome le da iconos vectoriales escalables que pueden personalizarse instantáneamente tamaño, color, sombra paralela, y cualquier cosa que pueda hacerse con el poder de hojas de estilo en cascada (Cascading Style Sheets [CSS, por sus siglas en inglés])

Roots.io

Un tema de arranque para Wordpress, le ayuda a construir mejores temas utilizando HTML5 Boilerplate, Bootstrap y Grunt

Piklist

Piklist is una extension rapida y concisa para el desarrollo efficaz Wordpress. Simplifica muchas de las tareas difíciles en WordPress y agrega funcionalidades extras que actualmente no se encuentran en el núcleo de WordPress

Grunt

A task runner. In one word: automation. The less work you have to do when performing repetitive tasks like minification, compilation, unit testing, linting, etc, the easier your job becomes

Bower

Un gestor de paquetes para el web. Las paginas web estan hechas de muchas cosas, frameworks, librerias, bienes, utilidades y arcoiris. Bower maneja todas estas cosas por usted

Adobe Color CC

Una applicacion Adobe que le permite probar, crear, y guardar distintos esquemas de colores, cada uno de los cuales consiste en un conjunto de cinco colores

Github

Colaboración poderosa, revisión de código, y manejo de código para una fuente abierta y proyectos privados

Wikipedia está dedicada a fomentar el crecimiento, el desarrollo y la distribución multilingüe, de contenido educativo para el público de forma gratuita
