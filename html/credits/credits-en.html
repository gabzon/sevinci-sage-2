Design & Development


Wordpress

WordPress is web software you can use to create a beautiful website or blog. We like to say that WordPress is both free and priceless at the same time.

Twitter bootstrap

Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

Jquery

jQuery is a fast, small, and feature-rich JavaScript library. jQuery has changed the way that millions of people write JavaScript.

Font Awesome

Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS.

Roots.io

A WordPress starter theme, helps you make better themes with HTML5 Boilerplate, Bootstrap, and Grunt.

Piklist

Piklist is a fast and concise plugin that encourages WordPress rapid development. It simplifies many of the more difficult tasks in WordPress and adds functionality not presently common in WordPress core.

Grunt

A task runner. In one word: automation. The less work you have to do when performing repetitive tasks like minification, compilation, unit testing, linting, etc, the easier your job becomes

Bower

A package manager for the web. Web sites are made of lots of things — frameworks, libraries, assets, utilities, and rainbows. Bower manages all these things for you.

Adobe Color CC

An Adobe application that lets individuals try out, create and save various colour schemes, each of which consists of a set of five colours.

Github

Powerful collaboration, code review, and code management for open source and private projects.
